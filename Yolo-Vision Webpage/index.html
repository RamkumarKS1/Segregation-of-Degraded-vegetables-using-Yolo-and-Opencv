Project Title:
Automated Segregation of Degraded Vegetables in Industrial Settings Using YOLO and OpenCV

Project Overview:

This project aims to automate the process of detecting and separating degraded or spoiled vegetables and fruits from fresh ones in an

industrial environment. It utilizes YOLO (You Only Look Once)—a real-time object detection algorithm—and OpenCV, a powerful image

processing library, to analyze visual data from a camera or conveyor belt system.

Key Objectives:

  • Identify degraded produce based on visual signs like discoloration, bruises, or fungal growth.

  • Classify items as “fresh” or “degraded” using trained YOLO models.

  • Automate the segregation process using computer vision, reducing manual effort and human error.

  • Improve efficiency, hygiene, and quality control in food processing units or packaging plants.
